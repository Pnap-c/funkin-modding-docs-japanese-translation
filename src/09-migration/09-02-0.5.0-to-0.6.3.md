# v0.5.0 から v0.6.3 への移行

v0.5.0 から v0.6.3 への移行には、MOD への変更が必要です。

# API バージョンの更新

Eric は v0.6.0 への更新時にゲームの API バージョンをインクリメントするのをうっかり忘れてしまい、これが v0.6.3 で修正されました。
これにより、`_polymod_meta.json`ファイル内の`api_version`値がv0.6.3以上でない限り、MODは読み込まれなくなります。これにより、MOD制作者はテストを実施し、選択した配布プラットフォームで更新をリリースする前に、自身のMODが互換性があることを確認できます。ほとんどのMOD（特に最小限のスクリプティングのみを使用しているもの）は、期待通りに動作するために変更を必要としませんが、開発者は念のためプレイテストを行うべきでしょう。

# hxCodec から hxvlc への移行

動画再生に使用するライブラリが hxCodec から hxvlc に変更されました。これにより全体的なパフォーマンス向上と各種バグ修正が実現しましたが、動画ライブラリと直接連携するMODは動作しなくなります。Funkin' のカットシーン用組み込みシステムを利用するMODは影響を受けません。

以下の変更を行ってください：

```haxe
// BEFORE: Imports from the package `hxcodec`
import hxcodec.flixel.FlxVideoSprite;

// AFTER: Imports from the package `hxvlc`
import hxvlc.flixel.FlxVideoSprite;


// BEFORE: Callback to onTextureSetup
video.bitmap.onTextureSetup.add(() -> { ... });

// AFTER: Callback to onFormatSetup
video.bitmap.onFormatSetup.add(() -> { ... });


// BEFORE: Play a video by path.
video.play(videoPath);

// AFTER: Load a video by path, then play if load was successful.
// You can also run video.load() in advance before playing the video.
if (video.load(videoPath)) video.play();
```

# オプションメニューの変更点

オプションメニューの内部構造を小規模に再構築しました。`pages`リストを独立したクラスに移動したため、「設定」メニューへのアクセスに必要なコードが変更されています（主にカスタム設定の追加を目的としています）。

将来的には、MODへのカスタムユーザー設定の追加プロセスを標準化したいと考えていますが、それまでの間は必要な調整を行ってください：

```haxe
// BEFORE: Retrieve the value from the page Map.
if (Std.isOfType(currentState, OptionsState)) {
    var preferencesPage = currentState.pages.get("preferences");

    // Create a new option.
    prefs.createPrefItemCheckbox(...);
}

// AFTER: Retrieve the value from the page Map, which is now inside a Codex.
if (Std.isOfType(currentState, OptionsState)) {
    var preferencesPage = currentState.optionsCodex.pages.get("preferences");

    // Create a new option.
    prefs.createPrefItemCheckbox(...);
}
```

# ステッカーの変更点

v0.6.0 では、ゲーム内でのステッカーの使用方法が書き換えられました（v0.6.3 ではさらに改良された形で再実装されています）。既存のステッカーを提供するMODは、おそらく動作しなくなるでしょう。

新規または更新中のMODでステッカーの追加・削除・置換を行う場合は、[カスタムステッカーパックのドキュメント](../07-custom-sticker-packs/07-00-custom-sticker-packs.md)を参照してください。
