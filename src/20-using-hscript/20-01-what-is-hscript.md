# HScriptとは？

HScriptは、Polymodが『Friday Night Funkin'』にカスタム機能を提供するために使用するインタプリタ型スクリプト言語です。ゲームは`.hxc`ファイルで提供されるスクリプトを検出し、解釈して実行し、ゲーム内でカスタム機能を実行します。

この機能を利用するのは改造だけではありません。基本ゲーム自体も、アニメーションやカットシーンの再生、ステージ小道具の操作、さらにはカスタムゲームプレイの追加にスクリプトを使用しており、スクリプトなしでは不可能な迅速な反復開発を可能にしています。

HScriptはHaxeと非常に類似しており、ゲーム内で利用可能なほぼ全ての変数やクラスにアクセスできます。ただし以下の注意点があります：

- 各スクリプトクラスは、他のスクリプトクラスと一意の名前を持つ必要があります。同じ名前のスクリプトクラスが2つ存在する場合、あるいは異なるMOD間で同じ名前のスクリプトクラスが存在する場合、いずれか一方が完全に抑制され、予期せぬ動作を引き起こす可能性があります。
- プライベート変数は簡単にアクセス可能です。操作には十分注意してください。
- 特定のクラスはブラックリストに登録されており、悪意のあるスクリプトからユーザーを保護します。
- `abstract enum`（Haxe言語機能。Enumのように振る舞う値を提供しますが、実際にはStringやIntegerなどの別の型です）はアクセス不可です。使用したい`abstract enum`のコードを確認し、実際の基盤となる値を見つけて代わりに使用してください。
- `abstract`（既存クラスに追加メソッドを適用するHaxe言語機能）はアクセス不可です。これらの値を操作するには何らかの回避策を見つける必要があります。
